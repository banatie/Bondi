英語圏。アメリカ。


概要
カジュアルなアプリで、運転のスコアリングをしてくれるアプリです。スマホを車内のホルダーに固定した状態で使用してもらいます。マップアプリを見ながらでも使えるように、バックグラウンドで実行できるようにしたい。

裏概要
Is your wife a good driver?

背景
私は、妻と子供2人がいるのですが、妻が運転する際に、「急アクセルだなー」や「目の前信号なのに、無駄な加速してるなー」などを感じることがあるのですが、直接妻に言うと、妻が面白くないそうです。そこで、楽しく運転のスコアリングをして、この運転の中で良かったところや、改善するところを見たいです。
例えば、以下のようなフィードバックが欲しいです。
・⚪︎回急アクセルがありました
・⚪︎回急ブレーキがありました
・全体を通して、追いついたステアリングでした
基本的には、ポジティブなアプリにしたいので、ネガティブな表現はしないです。
急ブレーキなどの表示も、ポジティブな表現にしたいです。

UI
・シンプルなUIを採用したいです。
・起動後は、スタートボタンだけあるようにしたい。
・スタートボタンを押すと、計測中と表示して、かっこいい感じのグルグル回る感じのアニメーションを表示して、計測中であることをユーザーに伝えたい。
・点数が出るまでの計算時間は、ドラムロールを出します。計算がすぐ終わっても、ドラムロールは、5秒表示します。
・以下の評価の基準に合致した場合に、派手なアニメーションを表示します。評価が良ければ良いほど、かっこよくて派手なアニメーションを表示します。

機能
・スタートボタンを押すと、運転を開始します。
・運転中は、加速度センサーを使用して、急アクセルや急ブレーキを検知します。
・運転中は、コンパスを使用して、追いついたステアリングを検知します。
・運転中は、バックグラウンドで実行できるようにしたいです。
・運転が終了して、ユーザーが停止ボタンをおすと、運転のスコアリングを表示します。
・点数は、１００点満点で、運転のスコアリングを表示します。
・点数が、以下の文字とともに表示されます。

評価の基準
　・80点以下 -> OK
　・80点以上 -> Good
　・85点以上 -> Great
　・90点以上 -> Excellent
　・95点以上 -> Fantastic
　・100点以上 -> Oh My God!

・運転のスコアリングは、以下のようなフィードバックを表示します。
・⚪︎回急アクセルがありました
・⚪︎回急ブレーキがありました
・全体を通して、追いついたステアリングでした
・設定画面を設けて、GPSを許可できるようにします。
・GPSを許可した場合、走った道を簡単なマップでとともに表示して、通った場所の運転が良かったところで、どこが良かったかを色でフィードバックしてくれるようにします。
・もし渋滞が長かった場合は、まずは渋滞お疲れ様でした。と表示してほしい。
